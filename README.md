Descripción corta del proyecto
Este proyecto corresponde al Ejercicio Iniciación Unit-Test 4. Consiste en una aplicación básica desarrollada con el objetivo de implementar y probar funcionalidades mediante pruebas unitarias. El proyecto permite validar el correcto funcionamiento de los componentes o funciones principales, asegurando que cumplan con los requisitos definidos.

Requisitos previos

Navegador web moderno (Google Chrome, Microsoft Edge, Firefox, etc.)

Editor de código recomendado: Visual Studio Code

Extensión Live Server (opcional, pero recomendada)

Cómo ejecutar el proyecto localmente
Opción 1:

Descargar o clonar el proyecto.

Abrir la carpeta del proyecto en Visual Studio Code.

Buscar el archivo index.html.

Hacer doble clic sobre el archivo para abrirlo en el navegador.

Opción 2 (recomendada con Live Server):

Abrir el proyecto en Visual Studio Code.

Instalar la extensión Live Server si no está instalada.

Hacer clic derecho sobre index.html.

Seleccionar la opción "Open with Live Server".

Estructura del proyecto

index.html: Archivo principal de la interfaz.

css/: Carpeta que contiene los estilos del proyecto.

js/: Carpeta que contiene la lógica y funciones en JavaScript.

tests/ o archivo de pruebas: Contiene las pruebas unitarias implementadas.

README.md: Archivo con la documentación del proyecto.

Funcionalidades implementadas

Inicialización correcta del proyecto.

Implementación de funciones principales en JavaScript.

Integración de pruebas unitarias.

Validación de resultados mediante pruebas.

Ejecución del proyecto en navegador web.

Organización del código en estructura modular.
